# SOP 前端框架架构设计文档

## 功能架构设计：SOP (Scaffold Operating Platform)

---

## 一、功能需求

**功能名称**: SOP - Vue3 后台管理二次开发框架

**功能类型**: 组件库型框架 / 业务功能框架

**功能描述**: 打造一款开箱即用的 Vue3 后台管理二次开发框架，通过配置化组件、Hooks 封装、多套主题方案，帮助开发者快速搭建企业级后台系统。

**核心功能模块**:

1. **基础设施层**
   - CLI 项目脚手架 (`@sop/cli`)
   - 工程化配置 (TypeScript + ESLint + Prettier + Husky)
   - 路由系统 (Vue Router 4 + 动态路由 + 路由守卫)
   - 状态管理 (Pinia + 持久化插件)
   - HTTP 封装 (Axios + 拦截器)
   - 国际化 (Vue I18n)

2. **布局系统层**
   - 上下布局 (Top + Content)
   - 左右布局 (Sidebar + Content)
   - 混合布局 (Top + Sidebar + Content)
   - 菜单管理 (多级菜单、折叠、搜索)

3. **权限认证层**
   - 登录/注册页面
   - JWT 管理 (Token 获取、自动刷新、登出)
   - 路由权限 (基于角色的路由守卫)
   - 按钮权限 (`v-permission` 指令)

4. **组件封装层**
   - `sop-form` - JSON Schema 配置化表单 (基于 Element Plus)
   - `sop-table` - 配置化表格 (基于 Vxe-Table)
   - `sop-upload` - 上传组件
   - `sop-echarts` - 图表组件
   - `sop-search` - 搜索表单

5. **主题系统层**
   - 浅色/深色/高对比主题
   - 在线主题定制工具

## 二、技术选型

### 2.1 核心技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.3+ | 前端框架 |
| TypeScript | 5+ | 类型系统 |
| Vite | 5+ | 构建工具 |
| Element Plus | 2.4+ | UI 组件库 |
| Vxe-Table | 4.5+ | 表格组件 |
| Vue Router | 4+ | 路由管理 |
| Pinia | 2+ | 状态管理 |
| Axios | 1.6+ | HTTP 请求 |
| Vue I18n | 9+ | 国际化 |
| @icon-park/vue-next | latest | 图标库 |

### 2.2 工程化工具

| 工具 | 用途 |
|------|------|
| pnpm | 包管理 + Workspace |
| ESLint | 代码质量检查 |
| Prettier | 代码格式化 |
| Husky | Git 钩子 |
| lint-staged | 暂存区代码检查 |
| Commitlint | 提交规范检查 |

### 2.3 选型理由

1. **Vue 3.3+**: 成熟的组合式 API，更好的 TypeScript 支持，更好的性能
2. **Element Plus**: 最成熟的 Vue3 UI 组件库，生态完善
3. **Vxe-Table**: 功能强大的表格组件，支持虚拟滚动、大数据量处理
4. **Pinia**: Vue 官方推荐的状态管理方案，TypeScript 友好
5. **pnpm + Monorepo**: 适合多包管理，依赖共享，节省磁盘空间
6. **Git Submodule**: 支持 pc-template 独立仓库引用 sop 核心库
